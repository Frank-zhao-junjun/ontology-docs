# US-S15-U05：W-EPC-15~17 规则（约束 + 事件配对 + 守卫绑定）

| 字段 | 值 |
|------|-----|
| **所属 US** | [US-S15](../us/US-S15-epc-wepc-ext.md) |
| **状态** | 草稿 |
| **依赖** | U01（类型字段），U02~U04（linter 模式） |
| **预估文件** | `src/lib/business-epc-linter/index.ts`, `tests/unit/business-epc-linter.spec.ts` |

## 1. 目标（一句话）

在 `lintBusinessEpc` 中增加 W-EPC-15、W-EPC-16、W-EPC-17 三条规则，检查约束类型完整性和行为-事件配对一致性。

## 2. 范围

### In Scope

- **W-EPC-15**：E7（约束）步骤引用的要素缺少 `constraintType` → 警告（step 级别）
- **W-EPC-16**：EPC 有 E2 行为步骤但无 E3 事件步骤 → 警告（EPC 级别）
- **W-EPC-17**：EPC 有 E7 guard 约束但无已绑定状态机的 E2 步骤 → 警告（EPC 级别）
- 对应单元测试

### Out of Scope

- WarningCenter UI（U06）

## 3. 技术设计

### W-EPC-15：约束类型完整性（step 级别）

在 `inspectConfirmedEpcStep` 中：

```typescript
if (step.elementRef.dimension === 'E7' && meta && !meta.constraintType) {
  pushWarning(warnings, {
    ruleId: 'W-EPC-15',
    message: `步骤「${step.name}」引用的约束要素「${meta.name}」未指定约束类型`,
    moduleKind: 'EPC',
    moduleId: epc.id,
    epcId: epc.id,
    stepId: step.id,
    elementId: step.elementRef.elementId,
  });
}
```

### W-EPC-16：Transition-Event 配对（EPC 级别）

在 EPC 级扫描中，检查是否同时存在 E2 和 E3 步骤：

```typescript
const hasE2 = steps.some((s) => s.elementRef?.dimension === 'E2');
const hasE3 = steps.some((s) => s.elementRef?.dimension === 'E3');
if (hasE2 && !hasE3) {
  pushWarning(warnings, {
    ruleId: 'W-EPC-16',
    message: `EPC「${epc.name}」有关联行为要素但缺少事件要素`,
    moduleKind: 'EPC',
    moduleId: epc.id,
    epcId: epc.id,
  });
}
```

### W-EPC-17：Guard-Action 守卫绑定（EPC 级别）

检查 E7 guard 约束是否需要匹配的已绑定状态机的 E2 行为步骤：

```typescript
const hasE7Guard = steps.some((s) => {
  if (s.elementRef?.dimension !== 'E7') return false;
  const em = metaById.get(s.elementRef.elementId);
  return em?.constraintType === 'guard';
});
const hasE2WithSM = steps.some((s) => {
  if (s.elementRef?.dimension !== 'E2') return false;
  const em = metaById.get(s.elementRef.elementId);
  return Boolean(em?.stateMachineId);
});
if (hasE7Guard && !hasE2WithSM) {
  pushWarning(warnings, {
    ruleId: 'W-EPC-17',
    message: `EPC「${epc.name}」有 E7 守卫约束但缺少已绑定状态机的 E2 行为步骤`,
    moduleKind: 'EPC',
    moduleId: epc.id,
    epcId: epc.id,
  });
}
```
